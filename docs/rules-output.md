# OUTPUT RULES (STRICT)

Copilot must keep all responses extremely concise.

Allowed output:
- Filenames changed
- A short description of the change
- Or: "Item complete."

Forbidden output:
- Summaries
- Multi-paragraph responses
- Explanations
- Architecture discussions
- Spec restatements
- Checklist restatements
- Diffs (unless asked)
- Large code blocks (unless asked)
- Reasoning (unless asked)

If the Orchestration Pipeline succeeds:
Respond with: "Item complete."
